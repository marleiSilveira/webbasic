Introdução ao html

O que é HTML?

HTML (Hyper Text Markup Language) é a linguagem de marcação padrão para criar páginas da Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. HyTime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo.

Características

  • HTML descreve a estrutura de uma página da Web
  • HTML consiste em uma série de elementos
  • Os elementos HTML informam ao navegador como exibir o conteúdo
  • Elementos HTML são representados por tags
  • As tags HTML rotulam partes do conteúdo, como "cabeçalho", "parágrafo", "tabela" e assim por diante
  • Os navegadores não exibem as tags HTML, mas as usam para renderizar o conteúdo da página

Tags

Tags HTML são nomes de elementos entre colchetes angulares. As Tags no documentohtml são lidas pelo navegador. Element é uma tag completa com abertura <tag> e fechamento </tag>. Um Element pode ter vários atributos, o Attribut é usado para mudar o valor do atributo. As tags HTML vêm em pares como <p> e </p> ou vazia sem tag de fechamento como <br>. A primeira tag em um par é a tag inicial ou de abertura, a segunda tag é a tag finalou de fechamento. A tag final é escrita como a tag inicial, mas com uma barra inserida antes do nome da tag.

O objetivo de um navegador da web (Chrome, Edge, Firefox, Safari) é ler documentos HTML e exibi-los. O navegador não exibe as tags HTML, mas as utiliza para determinar como exibir o documento.

Existem muitas tags, para saber mais visite lista de tags.

De acordo com o padrão HTML5 o <html>, o <body>e a <head> tag podem ser omitidos.

Descrição das tags:

  • <html> e </html>: Tag utilizada pelo navegador para identificar que o documento em questão trata-se de um HTML e não de texto puro. Entre e devem estar todas as outras tags, as informações e todo o conteúdo.
  • <head> e </head>: início e o fim do cabeçalho do documento, onde devem ser inseridas outras tags com dados e informações que não são exibidos na tela do navegador, mas utilizados por ele para fins diversos.
  • <title> e </title>: sempre contida entre e define o título do documento a ser exibido pela barra de títulos do navegador.
  • <body> e </body>: corpo do documento. Entre e está o que vai, efetivamente, ser exibido na tela do navegador.

A configuração do estilo de um elemento HTML pode ser feita com o style atributo. O style atributo HTML possui a seguinte sintaxe :

A property é uma propriedade CSS. O value é um valor CSS.

Quando um navegador lê uma marcador/folha de estilo, ele formata o documento de acordo com ela. Há três maneiras de inserir uma folha de estilo:

Folha de Estilo Externa

Uma folha de estilo externa é ideal quando o estilo é aplicado em muitas páginas. Com uma folha de estilo externa, você pode mudar a aparência de um sítio Web inteiro pela mudança de um arquivo. Cada página deve vincular-se à folha de estilo usando a tag <link>. A tag <link> vai dentro da seção head.

Folha de Estilo Interna

Uma folha de estilo interna deve ser usada quando um documento específico tem um estilo único. Você define estilos internos na seção head com a tag <style>.

Estilos Local (inline)

Um estilo local (inline) deve ser usado quando um estilo único deve ser aplicado a uma única ocorrência de um elemento. Para usar estilos locais (inline) você usa o atributo de estilo na tag relevante. O atributo de estilo pode conter qualquer propriedade CSS. Tipos de estilo de elementos HTML

  • style: atributo para estilizar.
  • background-color: propriedade CSS que define a cor de plano de fundo.
  • color: propriedade CSS define a cor do texto.
  • font-family: propriedade CSS define a fonte a ser usada.
  • font-size: propriedade CSS define o tamanho do texto.
  • text-align: propriedade CSS define o alinhamento horizontal do texto.

Cabeçalhos (Head)

O elemento head contém informação geral, também chamado meta-informação, sobre um documento. Meta significa "informação sobre". Meta-dados significa informação sobre dados, ou que meta-informação significa informação sobre informação.

Os metadados não são exibidos. Os metadados geralmente definem o título do documento, o conjunto de caracteres, os estilos, os scripts e outras metainformações. As seguintes tags descrevem metadados: <title>, <style>, <meta>, <link>, <script>, e <base>.

Imagens

Na HTML, as imagens são definidas com a tag <img>. É uma tag é vazia, com atributos e sem tag de fechamento.

Para exibir uma imagen em uma página, necessário usar o atributo src. Src significa "source" [fonte]. O atributo alt serve para adicionar uma descrição à imagem, valor do atributo src é a URL da imagem a ser exibida na página de acordo com a sintaxe: <img src="url" alt="Uma descrição da imagem">> Se um navegador não conseguir encontrar uma imagem, ele exibirá o valor do alt atributo. O alt atributo é obrigatório. Uma página da web não será validada corretamente sem ela.

Exemplo:

<img src="img_girl.jpg" alt="Girl in a jacket">

Imagens em outra pasta

Se não especificado, o navegador espera encontrar a imagem na mesma pasta que a página da web.

No entanto, é comum armazenar imagens em uma subpasta. Você deve incluir o nome da pasta no srcatributo:

Exemplo:

<img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

Imagens em outro servidor

As imagens podem ser acessadas de qualquer endereço da web:

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">

Imagens animadas

O HTML permite GIFs animados:

Exemplo:

<img src="programming.gif" alt="Computer Man" style="width:48px;height:48px;">

Links

Os links permitem que os usuários cliquem no caminho de uma página para outra. Podem ser texto , imagens ou um elemento html.

Hiperlinks

Links HTML são hiperlinks. Você pode clicar em um link e pular para outro documento. Quando você move o mouse sobre um link, a seta do mouse se transforma em uma pequena mão.

Sintaxe

Os hiperlinks são definidos com a <a> tag HTML.

Exemplo: <a href="https://www.w3schools.com/html/">Visit our HTML tutorial</a> O href atributo especifica o endereço de destino ( https://www.w3schools.com/html/ ) do link.

O texto do link é a parte visível (visite nosso tutorial em HTML). Ao clicar no texto do link, você será enviado para o endereço especificado.

Links Locais

O exemplo anterior usou um URL absoluto (um endereço da Web completo). Um link local (link para o mesmo site) é especificado com um URL relativo (sem https: // www ....).

O atributo de destino <target>

O target atributo especifica onde abrir o documento vinculado. O target atributo pode ter um dos seguintes valores:

  • _blank - Abre o documento vinculado em uma nova janela ou guia.
  • _self - Abre o documento vinculado na mesma janela / guia em que foi clicado (isso é o padrão).
  • _parent - Abre o documento vinculado no quadro pai.
  • _top - Abre o documento vinculado no corpo inteiro da janela.
  • framename - Abre o documento vinculado em um quadro nomeado.

Este exemplo abrirá o documento vinculado em uma nova janela / guia do navegador:

Exemplo: <a href="https://www.w3schools.com/" target="_blank">Visit W3Schools!</a>

Links HTML - Crie um marcador

Marcadores HTML são usados ​​para permitir que os leitores pulem para partes específicas de uma página da Web. Os marcadores podem ser úteis se uma página da web for muito longa. Para criar um marcador - primeiro crie o marcador e adicione um link a ele. Quando o link é clicado, a página rola para baixo ou para cima até o local com o marcador.

Elementos: id = " value " - para definir indicadores em uma página. href = "# value "- para vincular ao marcador.

Exemplo:

Primeiro, crie um marcador com o id tributo: <h2 id="C4">Chapter 4</h2>

Em seguida, adicione um link ao marcador ("Ir para o capítulo 4"), na mesma página:

Exemplo:

<a href="#C4">Jump to Chapter 4</a>

Você também pode adicionar um link a um marcador em outra página:

<a href="html_demo.html#C4">Jump to Chapter 4</a>
