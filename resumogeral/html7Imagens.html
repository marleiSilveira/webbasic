Imagens

Na HTML, as imagens são definidas com a tag <img>. É uma tag é vazia, com atributos e sem tag de fechamento.

Para exibir uma imagen em uma página, necessário usar o atributo src. Src significa "source" [fonte]. O atributo alt serve para adicionar uma descrição à imagem, valor do atributo src é a URL da imagem a ser exibida na página de acordo com a sintaxe: <img src="url" alt="Uma descrição da imagem">> Se um navegador não conseguir encontrar uma imagem, ele exibirá o valor do alt atributo. O alt atributo é obrigatório. Uma página da web não será validada corretamente sem ela.

Exemplo:

Código Resultado

<img src="img_girl.jpg" alt="Girl in a jacket">

Imagens em outra pasta

Se não especificado, o navegador espera encontrar a imagem na mesma pasta que a página da web.

No entanto, é comum armazenar imagens em uma subpasta. Você deve incluir o nome da pasta no srcatributo:

Exemplo:

<img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

Imagens em outro servidor

As imagens podem ser acessadas de qualquer endereço da web:

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">

Imagens animadas

O HTML permite GIFs animados:

Exemplo:

<img src="programming.gif" alt="Computer Man" style="width:48px;height:48px;">

Nos exemplos citados observe o que há em comum:

  • Use o elemento HTML para definir uma imagem
  • Use o src atributo HTML para definir o URL da imagem
  • Use o alt atributo HTML para definir um texto alternativo para uma imagem, se ela não puder ser exibida
  • Use o HTML widthe os height atributos para definir o tamanho da imagem
  • Use o CSS widthe as heightpropriedades para definir o tamanho da imagem (como alternativa)

Carregar imagens leva tempo. Imagens grandes podem diminuir a velocidade da sua página. Use imagens com cuidado.
